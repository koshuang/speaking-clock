# Persona: 家長

## 基本資料

| 項目 | 內容 |
|------|------|
| 名稱 | 美玲媽媽 / 志偉爸爸 |
| 年齡 | 35-45 歲 |
| 角色 | 國小學童家長 |
| 裝置 | 智慧型手機、家用平板（共用） |

## 人物背景

美玲是一位職業婦女，有一個小學二年級的兒子和一個幼稚園的女兒。每天下班後要處理孩子的功課、生活作息、才藝接送。她希望孩子能養成自律的習慣，但常常因為時間催促而和孩子起衝突。她正在尋找能幫助孩子建立時間觀念的工具，同時減少自己「一直唸」的負擔。

## 日常挑戰

| 時段 | 挑戰 |
|------|------|
| 早晨 | 催孩子起床、催刷牙、催吃早餐、催出門 |
| 放學後 | 催寫功課、限制 3C 時間、催洗澡 |
| 晚上 | 催上床睡覺、處理拖延 |
| 假日 | 作息混亂、兄弟姊妹搶平板 |

## 特徵與行為模式

### 心理特點
- 希望孩子自律，但不知道如何放手
- 對「又要唸孩子」感到疲憊
- 擔心孩子 3C 成癮
- 在意孩子的學習狀況和作息

### 使用習慣
- 習慣用手機管理家庭事務
- 會研究育兒相關的 App
- 願意為好用的工具付費
- 重視隱私和安全

### 教養觀念
- 希望培養孩子的自主能力
- 想用正向鼓勵取代責罵
- 相信習慣需要慢慢養成
- 希望親子關係和諧

## 痛點 (Pain Points)

1. **無止盡的催促**
   - 每天要催好幾次，說到生氣
   - 孩子對家長的聲音「免疫」了

2. **3C 時間控制困難**
   - 說好 30 分鐘，總是變成 1 小時
   - 強制收走會引發大衝突

3. **作息不固定**
   - 假日睡到中午，週一早上叫不起來
   - 晚上不睡，早上不起

4. **兄弟姊妹協調**
   - 多個孩子作息不同，需要分別管理
   - 共用裝置的時間分配

5. **自己也很忙**
   - 沒辦法時時刻刻盯著孩子
   - 需要工具代替部分提醒工作

6. **教養一致性**
   - 夫妻標準不一
   - 阿公阿嬤來會打亂規矩

## 需求 (Needs)

### 核心需求

| 優先級 | 需求 | 說明 |
|--------|------|------|
| P0 | 代替催促 | 用語音報時取代家長不斷唸 |
| P0 | 作息建立 | 幫助孩子建立固定的作息時間表 |
| P1 | 3C 時間管理 | 設定使用時間，時間到會提醒 |
| P1 | 任務完成追蹤 | 知道孩子完成了哪些事 |
| P2 | 多孩子管理 | 不同孩子不同設定 |
| P2 | 正向回饋機制 | 鼓勵孩子主動完成任務 |

### 功能需求對應

| 需求 | 現有功能 | 缺少/待改進 |
|------|----------|-------------|
| 代替催促 | ✅ 語音報時、任務提醒 | - |
| 作息建立 | ✅ 待辦清單、計時器 | 可增加「作息模板」 |
| 3C 時間管理 | ✅ 任務計時器 | 可增加「螢幕時間」類別 |
| 任務完成追蹤 | ✅ 完成標記 | 可增加每日/每週報告 |
| 多孩子管理 | ❌ 無 | 需要多帳號/設定檔功能 |
| 正向回饋 | ✅ 慶祝動畫、鼓勵語 | 可增加累積獎勵機制 |

## 使用情境 (Use Cases)

### 情境 1：放學後作息
> 美玲預先設定好放學後的待辦：「16:30 點心時間」「17:00 寫功課」「18:00 自由時間」「19:00 晚餐」。她不在家時，報時器會自動提醒孩子，減少阿嬤需要催促的次數。

### 情境 2：3C 時間約定
> 孩子完成功課後，可以玩 30 分鐘平板。美玲設定「玩平板 30 分鐘」的計時器，時間到會語音提醒。孩子知道這是規則，比媽媽說的更容易接受。

### 情境 3：睡前流程
> 晚上 8:30，報時器播報「現在是晚上 8 點 30 分，準備洗澡囉」。待辦清單顯示：洗澡 → 刷牙 → 準備書包 → 睡覺故事。孩子依序打勾，培養睡前習慣。

### 情境 4：假日作息維持
> 週末美玲也維持基本的報時設定：「9:00 起床」「12:00 午餐」「22:00 睡覺」。不會像平日那麼緊湊，但維持基本作息框架。

### 情境 5：查看完成狀況
> 美玲下班回家，打開報時器查看今天的「已完成任務」。發現孩子有確實完成功課和練琴，給予口頭讚美，強化正向循環。

## 選擇此產品的原因

### 比較其他方案

| 方案 | 優點 | 缺點 |
|------|------|------|
| 手機鬧鐘 | 簡單 | 只有聲音，沒有任務管理 |
| 計時器 App | 精確 | 沒有語音，孩子不會看 |
| 智慧音箱 | 語音互動 | 設定複雜，功能太多 |
| **語音報時器** | 語音提醒 + 任務管理 + 兒童友善 | - |

### 選擇關鍵因素
1. **語音報時**：孩子不需要盯著螢幕
2. **兒童模式**：介面友善，孩子能自己操作
3. **任務管理**：不只報時，還能管理待辦
4. **正向回饋**：有鼓勵機制，孩子願意使用
5. **免費/便宜**：不需要額外硬體

## 設計建議

### 初次設定引導
- 提供「快速設定」選項
- 詢問孩子年齡，自動推薦適合模式
- 預設作息模板可一鍵套用

### 家長控制功能
- 可鎖定設定，避免孩子自行修改
- 設定每日任務模板（平日/假日）
- 查看任務完成歷史紀錄

### 親子共用設計
- 簡單模式：孩子操作
- 設定模式：需要密碼/家長操作
- 清楚區分「孩子能動」和「家長才能動」的部分

### 通知設計
- 主動通知家長當日完成狀況（可選）
- 提醒家長調整設定（如孩子睡覺時間到了）
