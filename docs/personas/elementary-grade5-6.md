# Persona: 小學高年級學生（5-6年級）

## 基本資料

| 項目 | 內容 |
|------|------|
| 名稱 | 小雅 |
| 年齡 | 11-12 歲 |
| 年級 | 小學五至六年級 |
| 裝置 | 自己的手機、平板、電腦 |

## 人物背景

小雅是小學六年級學生，即將面臨國中升學。課業壓力明顯增加，除了學校功課，還有補習班和才藝課。她開始有自己的社交生活（班級群組、朋友約會），時間變得更加緊湊。她有能力管理時間，但常因事情太多而感到焦慮，或因優先順序判斷錯誤而手忙腳亂。

## 一日作息（平日）

| 時間 | 活動 |
|------|------|
| 06:30 | 起床、盥洗 |
| 07:00 | 吃早餐、整理書包 |
| 07:30 | 出門上學 |

| 22:30 | 上床睡覺 |

## 特徵與行為模式

### 認知發展
- **時間概念成熟**：完全會看時鐘，能理解時間規劃
- **抽象思維能力**：能做較長期的計劃（一週內）
- **專注力穩定**：可專注 45-60 分鐘
- **自我認同發展**：在意形象、同儕評價

### 行為特點
- 課業與課外活動並重，行程緊湊
- 開始使用社群媒體，容易分心
- 有較強的自主意識，不喜歡被控制
- 會為喜歡的事情熬夜（看小說、追劇）
- 能自己規劃，但執行力起伏大

### 情緒特點
- 容易因課業壓力感到焦慮
- 時間不夠用時會煩躁
- 希望被當成獨立個體
- 完成困難任務後很有成就感

## 痛點 (Pain Points)

1. **時間永遠不夠**
   - 功課、補習、才藝、社交全部擠在一起
   - 常常熬夜趕功課

2. **優先順序混亂**
   - 先做喜歡的事，困難的事一直拖
   - 考試前才臨時抱佛腳

3. **專注力分散**
   - 寫功課時會忍不住看手機訊息
   - 社群通知讓注意力中斷

4. **壓力管理困難**
   - 多重 deadline 重疊時會焦慮
   - 不知道如何有效休息

5. **作息混亂**
   - 假日睡到中午，平日精神不好
   - 睡前滑手機影響睡眠

## 需求 (Needs)

### 核心需求

| 優先級 | 需求 | 說明 |
|--------|------|------|
| P0 | 多任務管理 | 同時追蹤多項待辦和 deadline |
| P0 | 時間區塊規劃 | 將時間切分給不同任務 |
| P1 | 專注模式 | 幫助阻隔干擾，維持專注 |
| P1 | 休息提醒 | 讀書太久時提醒休息 |
| P2 | 進度視覺化 | 看到整體完成進度減少焦慮 |
| P2 | 週計劃檢視 | 一眼看到一週的安排 |

### 功能需求對應

| 需求 | 現有功能 | 缺少/待改進 |
|------|----------|-------------|
| 多任務管理 | ✅ 待辦清單 | 可增加 deadline/到期日欄位 |
| 時間區塊規劃 | ✅ 任務計時器 | 可增加每日時間表檢視 |
| 專注模式 | ❌ 無 | 需要專注模式（隱藏干擾） |
| 休息提醒 | ⚠️ 報時功能 | 可增加「連續 50 分鐘提醒休息」 |
| 進度視覺化 | ✅ 進度條 | 可增加每日/每週統計 |
| 週計劃檢視 | ❌ 無 | 需要週曆/計劃檢視 |

## 使用情境 (Use Cases)

### 情境 1：考試週複習
> 下週有三科考試。小雅在報時器設定：「數學複習（60分鐘）」「社會重點整理（45分鐘）」「英文單字背誦（30分鐘）」。報時器依序計時，每完成一科就語音鼓勵，並在連續讀書 50 分鐘時提醒「休息 10 分鐘，伸展一下」。

### 情境 2：補習班作業管理
> 補習班老師出了三天後要交的作業。小雅在待辦清單新增任務，設定 deadline。報時器會在到期前一天、當天早上提醒「補習班數學作業明天要交喔」。

### 情境 3：週末時間規劃
> 週六早上，小雅打開報時器規劃一天：早上練鋼琴 1 小時、下午和朋友出去、晚上做作業。報時器提醒她每個時段該做什麼，避免玩太久忘記正事。

### 情境 4：睡前作息
> 晚上 10 點，報時器提醒「該準備睡覺了，明天還要早起」。即使小雅還想滑手機，報時器在 10:15 再次提醒「已經超過預定睡覺時間 15 分鐘囉」。

## 家長期望

1. **課業與休閒平衡**：確保有足夠時間完成功課，但也有適當休息
2. **減少 3C 沉迷**：特別是社群媒體和追劇時間
3. **自主但有邊界**：給予自主權，但有工具幫助維持底線
4. **升學準備**：養成好的時間管理習慣，為國中做準備

## 設計建議

### 介面設計
- 簡潔專業的風格（不幼稚）
- 支援深色模式（晚上使用）
- 資訊密度較高，一眼看到重要資訊
- 可自訂主題顏色

### 語音設計
- 語速正常或稍快
- 語調中性專業
- 可關閉兒童模式的語音效果

### 互動設計
- 快速新增任務（最少步驟）
- 拖曳排序調整優先順序
- 批次完成/刪除功能
- 快捷操作（鍵盤快捷鍵）

### 功能設計
- 到期日/deadline 設定
- 週計劃/月計劃檢視
- 連續專注時間統計
- 學習效率報告
- 專注模式（極簡介面）

## 與其他年齡層差異

| 面向 | 1年級 | 3-4年級 | 5-6年級 |
|------|-------|---------|---------|
| 時間觀念 | 薄弱 | 發展中 | 成熟 |
| 自主程度 | 需要引導 | 半自主 | 高度自主 |
| 任務複雜度 | 單一任務 | 多任務 | 多任務 + deadline |
| 介面需求 | 大按鈕、圖示 | 清楚易用 | 資訊密度高 |
| 語音需求 | 慢速、親切 | 正常 | 可選關閉 |
| 回饋機制 | 動畫、音效 | 成就感 | 數據統計 |
