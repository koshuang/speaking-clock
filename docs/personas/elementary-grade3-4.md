# Persona: 小學中年級學生（3-4年級）

## 基本資料

| 項目 | 內容 |
|------|------|
| 名稱 | 小明 |
| 年齡 | 9-10 歲 |
| 年級 | 小學三至四年級 |
| 裝置 | 自己的平板（家長設有使用限制）、家用電腦 |

## 人物背景

小明是小學三年級學生，已經適應了學校生活，開始有更多的課業和課外活動。他會看時鐘了，但對於時間管理的概念還不成熟。他開始有自己的興趣（電玩、足球），常常因為太投入而忘記時間。家長希望他能開始學習自主管理時間，而不是事事依賴提醒。

## 特徵與行為模式

### 認知發展
- **時間概念進步**：會看時鐘，知道幾點幾分，但對「還要多久」的估算不準
- **抽象思維萌芽**：開始理解「規劃」的概念，但執行力不足
- **專注力提升**：可專注 30-40 分鐘
- **自我意識增強**：開始在意同儕，希望被當作「大孩子」

### 行為特點
- 功課量增加，需要更多時間完成
- 開始有課外活動（才藝班、運動隊）
- 對 3C 產品興趣濃厚，容易沉迷
- 會討價還價「再 5 分鐘」
- 開始有責任感，但不穩定

### 情緒特點
- 不喜歡被當小孩對待
- 希望有自主權和選擇權
- 對無聊的事情容易失去耐心
- 完成挑戰後有成就感

## 痛點 (Pain Points)

1. **時間估算偏差**
   - 總覺得「還有很多時間」
   - 功課寫到一半才發現快沒時間了

2. **多工困難**
   - 功課、才藝班、休閒時間難以平衡
   - 常常顧此失彼

3. **拖延習慣**
   - 知道該做功課，但想先玩一下
   - 「等一下」變成「再等一下」

4. **3C 成癮傾向**
   - 玩遊戲/看影片很難自己停止
   - 需要外部提醒才會結束

5. **轉換抗拒**
   - 從玩樂轉換到功課時會抱怨
   - 需要緩衝時間適應

## 需求 (Needs)

### 核心需求

| 優先級 | 需求 | 說明 |
|--------|------|------|
| P0 | 任務時間管理 | 知道每個任務要花多少時間 |
| P0 | 預警提醒 | 在時間快到前先提醒，有心理準備 |
| P1 | 進度追蹤 | 看到自己完成了多少、還剩多少 |
| P1 | 自主設定 | 可以自己設定部分提醒（培養自主） |
| P2 | 成就系統 | 累積完成任務的成就感 |
| P2 | 彈性調整 | 可延長或縮短任務時間 |

### 功能需求對應

| 需求 | 現有功能 | 缺少/待改進 |
|------|----------|-------------|
| 任務時間管理 | ✅ 任務計時器 | - |
| 預警提醒 | ✅ 剩餘時間播報 | 可增加「還剩 5 分鐘」特別提醒 |
| 進度追蹤 | ✅ 進度條 | 可增加每日/每週完成統計 |
| 自主設定 | ⚠️ 部分 | 需要「兒童可用」的簡化設定介面 |
| 成就系統 | ⚠️ 鼓勵語 | 可增加累積徽章/等級 |
| 彈性調整 | ❌ 無 | 需要延長/縮短計時功能 |

## 使用情境 (Use Cases)

### 情境 1：功課時間規劃
> 小明放學回家，打開報時器看到待辦清單：數學習作（30分鐘）、國語造句（20分鐘）、聯絡簿簽名。他選擇先做數學，按下開始，報時器播報「下午 5 點 30 分，現在是數學習作時間，共 30 分鐘」。中間聽到「還剩 10 分鐘」的提醒，他加快速度完成。

### 情境 2：遊戲時間控制
> 寫完功課後，小明想玩 Switch。他自己設定「玩遊戲 40 分鐘」的計時器。報時器會在剩餘 10 分鐘、5 分鐘時提醒他，讓他有心理準備。時間到時播報「遊戲時間結束囉，今天玩得開心嗎？」。

### 情境 3：課外活動提醒
> 週三下午有足球隊練習。報時器在 16:00 提醒「還有 30 分鐘要去足球練習，記得準備球鞋和水壺」。小明有足夠時間換衣服、準備裝備。

## 家長期望

1. **培養自主管理**：減少盯功課的時間，讓孩子學會自己安排
2. **建立時間觀念**：知道事情需要花多少時間，學會估算
3. **控制 3C 使用**：有工具幫助孩子遵守使用時間約定
4. **減少親子衝突**：用客觀的工具代替家長的催促

## 設計建議

### 介面設計
- 清楚但不幼稚的視覺風格
- 數據化的進度顯示（百分比、剩餘時間）
- 可自訂介面主題
- 適度保留兒童模式元素（可選）

### 語音設計
- 語速正常（不需要刻意放慢）
- 語調友善但不幼稚
- 可選擇不同語音角色

### 互動設計
- 提供「再 5 分鐘」的延長選項（有次數限制）
- 倒數計時的視覺化（進度條 + 數字）
- 任務完成的成就回饋

### 功能設計
- 每週時間使用報告（給家長）
- 任務模板可自訂
- 預設適合年齡的任務時長
- 每日目標設定
